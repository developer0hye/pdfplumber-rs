## Codebase Patterns
- `ExtractOptions` in `pdfplumber-core/src/error.rs` controls pipeline behavior. Add new options there and apply in `Pdf::page()` in `pdfplumber/src/pdf.rs`.
- Cross-validation golden data is generated from Python pdfplumber without normalization or dedup. When comparing, disable features Python doesn't apply by default.
- `Page::from_extraction()` is the internal constructor — dedup/normalization happen in `Pdf::page()` before calling it.
